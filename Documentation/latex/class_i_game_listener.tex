\doxysubsection{Detailed Description}
The interface for receiving game-\/related events and notifications. 

This interface defines methods to be implemented by classes that wish to receive notifications about various events that occur during a game, such as game over, check, pawn evolution, tie requests, piece movement, captures, restarts, and timer ticks.
